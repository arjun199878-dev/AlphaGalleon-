# AlphaGalleon UI Design Prompt

**Role:** Senior Mobile UI/UX Designer (Specializing in FinTech/Investment Apps).

**Context:**
We are building "AlphaGalleon," an institutional-grade personal investment banker app. The target audience is retail investors who want professional, high-signal intelligence without the clutter.

**Goal:**
Design a premium, clean, and trustworthy mobile interface (React Native style).

**Key Screens & Requirements:**

1.  **Bottom Navigation (Tab Bar):**
    -   **Home:** Dashboard with market indices (NIFTY/SENSEX), search bar, and a feed of "Most Bought" or "Trending" stocks.
    -   **Tools (The Intelligence):** A central hub grid for our AI modules:
        -   *The Architect* (Portfolio Builder).
        -   *The Doctor* (Portfolio Diagnosis).
        -   *Time Travel* (Backtesting).
    -   **Profile:** A clean account view (similar to Groww/Zerodha) showing avatar, balance, and settings list.

2.  **Visual Style (The "Institutional" Look):**
    -   **Reference:** Groww (Clean White), Zerodha Kite (Minimalist).
    -   **Primary Color:** #00D09C (Emerald/Teal Green for "Growth/Wealth").
    -   **Background:** Clean White (#FFFFFF) with subtle light gray (#F3F4F6) for grouping.
    -   **Typography:** Professional sans-serif (Inter/SF Pro). High legibility.
    -   **Cards:** Subtle shadow/elevation, rounded corners (8-12px).

3.  **Detailed Components:**
    -   **Stock Card:** Symbol icon, Company Name, Current Price (Bold), and Day Change (Green/Red).
    -   **Module Card (Tools):** Icon (e.g., Stethoscope for Doctor), Title, and short description.

**Output Required:**
Provide the High-Fidelity Mockups or detailed React Native style code for the **Home Screen**, **Tools Hub**, and **Profile Screen** following these institutional guidelines.
